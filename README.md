# Defender Security Tools

Standalone HTML tools for Microsoft Defender / ASR rollout work.

## Files

- `index.html` - landing page
- `asr-readiness.html` - ASR Readiness Dashboard
- `recommendation-analyzer.html` - Defender Recommendation Analyzer

## Usage

Open `index.html` locally or publish the repository with GitHub Pages.

All parsing happens locally in the browser. No data is uploaded to any server.
